なぜAI生成「無修正」コンテンツが可能なのか
構造的要因分析

1. オープンソースモデルのローカル実行
Stable Diffusionを筆頭とする画像生成AIはオープンソースで公開されており、ユーザーのローカルPC上で完全に制御可能。クラウドAPIと異なり安全フィルター（Safety Checker）を物理的に除去できるため、出力に制限がない。

2. 学習データの特性
SD 1.5はLAION-5B（約50億画像）から学習。このデータセットにはNSFWフィルタリングが不完全な画像が含まれており、モデルは元から無修正画像の生成能力を持つ。さらにコミュニティが明示的に無修正データセットでファインチューニングしたチェックポイントが数百種類存在。

3. LoRAによる制限解除メカニズム
Flux等の制限付きモデルでも、LoRA（Low-Rank Adaptation）と呼ばれる軽量な追加学習により安全フィルターの挙動を上書きできる。全パラメータの0.1-1%を変更するだけで、数MB〜数十MBのファイルでモデルの出力傾向を根本的に変更可能。

4. モザイク除去AIの並行進化
既存の修正済みコンテンツからモザイク/検閲領域を自動検出し、GAN（敵対的生成ネットワーク）で「推定復元」するツールが複数存在。これは厳密には「復元」ではなく「もっともらしい画像の生成」だが、視覚的に説得力のある結果を出力。

なぜ特定モデルがNSFW生成可能なのか
学習データ起因

SD 1.5のベースデータセットLAION-5Bには約50億画像が含まれ、そのNSFWフィルタリングは不完全。モデルは明示的な画像の生成パターンを暗黙的に学習済み。

コミュニティプロジェクト（Unstable Diffusion等）は3000万枚以上の明示的NSFWデータセットを構築し、チェックポイントのマージ/ファインチューニングに使用。

Safety Checker 除去

公式SDにはSafety Checkerが搭載されているが、これはモデル外部の後処理フィルターであり、モデル自体にNSFW抑制は組み込まれていない。

AUTOMATIC1111等のローカルUIはSafety Checkerをデフォルトで無効化。コード上で削除するだけで全制限が解除される。

マージ/ファインチューニング

複数チェックポイントの「マージ」により、あるモデルのNSFW能力と別モデルの品質を組み合わせる技法が一般的。

DreamBooth/LoRAによる追加学習で特定のスタイルや人物を学習させ、無修正の出力品質を向上させる。

Inpainting ワークフロー詳細
Inpainting の仕組みと手順

原理: 画像の指定領域（マスク）にのみランダムノイズを付加し、テキストプロンプトに基づいて再生成する技術。マスク外の領域は保持される。

手順:

1. 元画像をImg2Imgタブの「Inpaint」に読み込み

2. 再生成したい領域を白でマスク（ブラシツールで描画）

3. プロンプトに生成内容を記述（例: 特定の衣服、肌の質感等）

4. Denoising Strengthを調整: 0.4-0.7が最適範囲

5. 「Inpaint area: Only masked」で高精細化

6. 結果確認 → 不満足なら再度マスク調整して繰り返し

LoRA (Low-Rank Adaptation) 技術詳解
技術的仕組み

原理: 大規模モデルの重み行列 W を直接更新する代わりに、低ランクの分解行列 W + BA（B: d×r, A: r×k, r << min(d,k)）を追加学習。全パラメータの0.1-1%のみを学習するため、メモリ効率が極めて高い。

直感的理解: 数十GBのモデル全体を再学習せず、数MB〜数十MBの「パッチ」を当てるイメージ。複数のLoRAを同時に適用可能（スタイルLoRA + キャラLoRA + 制限解除LoRA等）。

NSFW文脈での意義: Flux等のベースモデルは安全フィルターが組み込まれているが、LoRAによりフィルター部分の重みを上書きし、制限を解除できる。これが「Uncensored LoRA」の仕組み。

AIモザイク除去技術
技術的仕組み

重要な前提: AIモザイク除去は「復元」ではなく「推定生成」。元の画像情報はモザイク処理で不可逆的に失われており、AIは文脈から「もっともらしい画像」を生成しているに過ぎない。

処理パイプライン:

1. 検出フェーズ: 物体検出モデル（YOLO, Mask R-CNN等）でモザイク/検閲領域を自動識別

2. セグメンテーション: 検出領域のピクセルレベルマスクを生成

3. 復元フェーズ: GAN（Generative Adversarial Network）による画像生成。Generator が「本物らしい」画像を生成し、Discriminator が真贋を判定する敵対的学習により品質を向上

4. 後処理: 境界のブレンディング、色調整、ノイズ除去

AnimateDiff 詳細
仕組み

アーキテクチャ: SD1.5の画像生成パイプラインにモーションモジュール（Temporal Transformer）を挿入。短い実写動画クリップから動きのパターンを学習し、静止画生成プロセスに時間的整合性を付与。

NSFW利点: SD1.5ベースのため、既存のNSFWチェックポイント（Realistic Vision、CyberRealistic等）と完全互換。モデルを差し替えるだけでNSFW動画が生成可能。

Deepfake / Undress AI 問題

技術: Stable Diffusion Inpaintingベースの衣服除去AI。実在人物の写真から性的画像を自動生成。

問題:

決済追跡: ブロックチェーン分析企業（Elliptic等）が暗号通貨決済の追跡により違法サービスの特定に協力。

事例1: わいせつポスター販売（2025/04）

被疑者: 愛知県北名古屋市・小売業 水谷智浩容疑者(44)ら20〜50代の男女4人

手口: 無料の画像生成AIに女性の体のデータを読み込ませ、「脚を開く」等のポーズ指示でAI美女画像を生成。ネットオークションに出品時はモザイク付きで掲載し、実際に送付したポスターは無修正。1枚数千円で「AI美女」「一点物」として販売。

処罰理由: 刑法175条「わいせつ図画頒布」— AI生成物であっても社会通念上わいせつと認められる画像を不特定多数に販売（頒布）した行為が構成要件を満たす。

動機: 「ポスター販売は原価が安く稼げる」というネット投稿を見て開始。約1年で売上1,000万円。

事例2: 抱き枕カバー転売（2025/06）

被疑者: 埼玉県新座市・自営業 46歳男性

手口: オークションサイトでAI生成裸体女性が印刷された抱き枕カバーを購入し、フリマサイトで「AI生成美女」として転売。自身は画像を生成していない（転売のみ）。

処罰理由: 刑法175条「わいせつ物頒布」— 自ら生成していなくても、わいせつ物を販売・頒布した行為自体が処罰対象。実在しない人物の画像でも適用された点が画期的。

動機: 「生活費のため。AIで作った画像の方が売れると思った」

売上: 2024年8月〜12月に数万円

事例3: 大学生のAI画像販売+賭博（2025/09）

被疑者: 神奈川県藤沢市・大学1年生 19歳男性

手口: 海外サイトからAI生成わいせつ画像を入手し、「AI美女」として約300点をネット出品。1点300〜2,000円で販売し約28万円を得た。売上金をオンラインカジノ（海外サイト）に入金し賭博。

処罰理由: わいせつ図画頒布 + 賭博罪の複合。2024年2月〜2025年7月に約60万円を賭け、約20万円の損失。

動機: 「賭博の賭け金にするために売った」と供述

事例4: 芸能人ディープフェイク（2025/10）

被疑者: 秋田市・会社員 横井宏哉容疑者(31)

手口: 無料の生成AIソフトに著名人の画像を学習させてわいせつ画像を作成。被害者は女性芸能人262人。月額制の有料サイトで約2万点を閲覧可能にし、2024年10月〜2025年9月に約120万円を売上。

処罰理由: 「わいせつ電磁的記録媒体陳列」— 有料でインターネット上で閲覧可能にした行為が「公然陳列」に該当。実在芸能人を対象としたため、名誉毀損・肖像権侵害も追及される可能性。

動機: ネット上の記事や動画で作り方を知ったと説明

事例5: AI生成児童ポルノ（2025/12）

被疑者: 元小学校教諭

容疑: 児童買春・ポルノ禁止法違反（AI生成女児の性的画像の所持）

処罰理由: AI生成物であっても児童ポルノに該当すると判断された全国初の事例。刑法175条ではなく、より重い児童ポルノ禁止法が適用。児童ポルノは「頒布」だけでなく「所持」自体が犯罪（2014年法改正以降）。

意義: 実在しない児童のAI生成画像でも児ポ法が適用されることが確認された。教職にある者の所持であり、社会的影響が大きい事例。

摘発パターンの分析

共通する処罰の鍵:

1. 頒布・公然陳列 — 全事例で販売・公開行為が要件。個人的な生成・保持のみでは刑法175条は適用されない（児ポ法除く）

2. 営利目的 — 全事例で金銭的利益を得ている（数千円〜1,000万円）

3. AI生成物 = わいせつ物 — 裁判所/検察は「AI生成であること」を免責事由と認めていない

4. 実在性は不問 — 事例2で確認（実在しない人物のAI画像でも適用）

2025年の傾向: 「全国初」の摘発が相次ぎ、捜査機関がAI生成わいせつ物への法適用を積極的に拡大。被害は芸能人から一般人・未成年者にまで拡大。卒業アルバムやSNS写真を悪用したいじめ事案も。

加害者プロファイル分析
類型1: 営利目的販売者

特徴: AI生成画像をポスター・グッズ化して販売。「原価が安く稼げる」動機。

事例: 水谷(44・小売業・1,000万円)、19歳大学生(28万円)、46歳自営業(数万円)

手法: フリマ/オークションサイトで「AI美女」として出品。モザイク付きで出品→無修正で納品。

リスク認識: 低い。「AIが作った画像だから問題ない」との認識が共通。

類型2: ディープフェイク作成者

特徴: 実在人物（芸能人・知人）の顔をAIで性的画像に合成。

事例: 横井(31・会社員・芸能人262人)、韓国10代グループ、豪Rotondo

動機: 「趣味・楽しみ」(Rotondo) から収益化(横井)まで幅広い。

被害者: 芸能人→一般人→同級生→卒業アルバム写真の被害者へと拡大。

類型3: AI-CSAM生成者

特徴: AI（主にStable Diffusion）で児童の性的画像を生成。最も重い刑罰対象。

事例: Anderegg(42・SE・13,000枚)、Tatum(児童精神科医・禁固40年)、Culmo(40)、日本元教諭

共通点: 専門技術者・教育者・医療従事者など社会的地位のある者が含まれる。

刑罰: 最も厳しく、禁固10-70年の範囲。「実在児童が存在しない」は減刑理由にならない傾向。
